P2P Downloader to communicate with the webservers of CICS.umass.edu
Everything should execute by running P2PDownloader.py
Objective of the project is to establish Peer-to-Peer connection's in order to download a file

TCP is used to exstablish connections with peers
UDP is used to communicate with Umass Server's to grab possible peer IP addresses. 
There is a limitation on the number of peers we can request in a given ammount of time.
